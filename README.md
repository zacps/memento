# MEMENTO

MEMENTO is a Python library for running computationally expensive experiments.

## Developing

Install dependencies:

```
$ pip install poetry
$ poetry install
```

To add dependencies:

```
$ poetry add (--dev) fancy_library
```

To run tests:

```
$ poetry run pytest
```

To run linters:

```
$ poetry run pytlint . && poetry run black .
```
